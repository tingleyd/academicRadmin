Overview

sparsereg: Function for fitting a Bayesian LASSOplus model for sparse models with uncertainty, facilitating the discovery of various types of interactions. Function takes a dependent variable, an optional matrix of (pre-treatment) covariates, and a (optional) matrix of categorical treatment variables. Includes correct calculation of uncertainty estimates, including for data with repeated observations.

sparseregIV: The function sparseregIV allows for estimation the treatment effect of an endogenous variable on an outcome, in the presence of a valid instrument. Up to two instruments are supported. Estimation occurs in two stages, just as in two stage least squares, except the function uses a high-dimensional, sparse, nonparametric model instead of ordinary least squares. The difference between these two values gives the desired treatment effect. See below for examples.

sparseregTE: Function for estimating a treatment effect using a nonparametric Bayesian LASSOplus model. Function takes a dependent variable, treatment variable, a matrix of (pre-treatment) covariates, and the desired amount to shift the treatment. Also handles data with repeated observations.

sparseregNP: Creates covariate x covariate tensor-basis and conducts sparse regression. Useful for observational studies with some covariates.

sparseregMediate: Function for estimating an mediation model using a nonparametric Bayesian LASSOplus model.

sparsereg

### sparsereg
set.seed(1)
n <- 200
p <- 5

## Exogenous treatments
treat  <- sample(c("a","b","c"), n,  replace = TRUE, pr = c(0.5, 0.25, 0.25))
treat2 <- sample(c("a","b","c","d"), n, replace = TRUE, pr = rep(0.25, 4))

## Xs
S <- diag(p)
S[S == 0] <- runif(p^2 - p)
S <- t(S) %*% S
X <- mvrnorm(n, mu = rep(0, p), Sigma = S)

## y as a function of main effects and interaction effects
# 3 + 2X + 2a - 2b - 2bX + 2cX
y.true <- 3 + X[,2]*2 + 
  (treat == "a")*10 + 
  (treat == "b")*(-2) + 
  X[,2]*(treat == "b")*(-2)+
  X[,2]*(treat2 == "c")*2

y <- y.true + rnorm(n, sd = 2)

## Fit a sparsereg model
fit    <- sparsereg(y, X, cbind(treat, treat2), scale.type = "TX")
fit.EM <- sparsereg(y, X, cbind(treat, treat2), scale.type = "TX", EM = TRUE)

save(fit, file = 'fit.RData')
save(fit.EM, file = 'fitEM.RData')
summary(fit)

## sparseregIV
set.seed(101)
n <- 100
p <- 5

S <- diag(p)
S[S == 0] <- runif(p^2 - p)
S <- t(S) %*%S
X <- mvrnorm(n, mu = rep(0, p), Sigma = S)
inst <- rnorm(n, sd = 1)
Sigma.IV <- matrix(0.1*c(1, 0.5, 0.5, 1), nrow = 2) 
errs <- mvrnorm(n, mu = c(0, 0), Sigma = Sigma.IV)
treat <- X[, 1]*inst + errs[, 1]
y <- X[, 2]*treat + errs[, 2]

## run sparseregIV
fit.iv <- sparseregIV(y = y, endog = treat, inst = inst, X = X)
